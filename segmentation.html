Medical Imaging • Python • ITK/VTK

Project Overview

Este projeto consistiu no desenvolvimento de uma Interface Gráfica (GUI) em Python para otimizar a análise comparativa de segmentações tridimensionais de estruturas anatómicas[cite: 125, 134]. O sistema combate a subjetividade do diagnóstico manual ao permitir a comparação visual e quantitativa direta entre diferentes observadores[cite: 124, 135].

🔍

Synchronized 3D Viewing

Visualização sincronizada de conjuntos de dados 3D através do toolkit VTK, garantindo o mesmo referencial anatómico em ambas as janelas[cite: 163, 166].

📊

Automated Metrics

Extração automática de parâmetros de intensidade (média, desvio padrão) e geométricos (volume, arredondamento, diâmetros de Feret) via ITK[cite: 167, 169, 170].

⚠️

Discrepancy Alerts

Geração de tabelas comparativas com alertas visuais automáticos sempre que a diferença entre observadores ultrapassa os 10%[cite: 135, 173].

💾

Data Export

Histórico de sessões integrado com capacidade de exportação de dados em formato DAT para análise estatística externa[cite: 127, 174].
