Project Overview

This project involved the development of a Python-based Graphical User Interface (GUI) designed to optimize the comparative analysis of 3D medical segmentations. By leveraging ITK (Insight Toolkit) and VTK, the system provides synchronized visualization and automated metric calculation, effectively tackling the subjectivity and inter-observer variability inherent in manual medical image segmentation.

Synchronized 3D Viewing

Implemented 3D render window synchronization where any interaction (rotation or zoom) in one observer view is instantly mirrored in the other, maintaining the same anatomical reference.

Advanced Metrics Engine

Automated extraction of intensity parameters (Mean, SD, Skewness, Kurtosis) and geometric features (Volume, Roundness, Feret Diameters) using the ITK framework.

Session & Data Logistics

Integrated session history module for tracking analysis progress and exporting comprehensive clinical data in .DAT format for external statistical validation.

Technical Stack & Implementation

  • Core Toolkit: Python with PyQt5 for GUI architecture and SimpleITK for volumetric data processing.
  • Visualization: VTK backend for high-performance 3D rendering and synchronized camera mapping.
  • Imaging Data: Optimized for Time-of-Flight Magnetic Resonance Angiography (TOF-MRA) datasets.
